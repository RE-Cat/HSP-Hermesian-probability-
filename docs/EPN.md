# 语气标音系统（最终版）

## 一、系统概述

这是一套用于标注口语中“怎么说”的符号系统，解决现有文字输入无法表达语气、情绪、声调、节奏等问题。

**括号规则：**
- `[ ]` 只用于注音符号
- `{ }` 包裹整段物体，可叠加
- `( )` 只用于循环和函数
- `< >` 只用于写参数
- 只在循环和函数同时出现时作为分隔符
- 所有间隔符号左右必须加空格。

## 三、语法规则

1. **注音符号包裹**  
   所有注音符号必须用 `[ ]` 包裹  
   ✅ 正确：`[ * ]`（不）  
   ❌ 错误：`*`（不）

2. **整段语气包裹**  
   整段语气用 `{ }` 包裹，可叠加  
   `{ 师傅别闹 }`  
   `{{ 师傅[^] 别闹 }}`  
   `{ & 师傅[^] 别闹 }`

3. **循环和函数**  
   循环：`( <+-> @ <3> )`（嗯……）  
   函数定义：`( ?! | *I | ;! <极度震惊> )`  
   函数调用：`( :极度震惊 )`（什么）！  
   循环+函数：`( <+-> @ <3> # :思考抖动 )`（嗯……）

4. **参数包裹**  
   参数用 `< >` 包裹  
   `<3>`  
   `<极度震惊>`  
   `<30>`

5. **强度符号位置**  
   强度符号 `I` / `W` 放在被修饰符号后面  
   ✅ 正确：`[ *I ]`  
   ❌ 错误：`[ I* ]`

6. **间隔符号空格**  
   所有间隔符号左右必须加空格  
   `|` 和 `#` 前后都要加空格  
   `[ * | ~ | + ]`  
   `( <+-> @ <3> # :思考抖动 )`

7. **符号自由组合**  
   `[ * | ~ | + ]`  
   `[ #<30> | << | ;* | ~: ]`

8. **作用范围**  
   - 单字精细控制：`[ ; ]` 你  
   - 多字精细控制：`[ ; ]`（你好）

9. **符号可逆序**  
   `:~` 与 `~:` 互为反向

10. **停顿符号独立存在**  
    `[ / ]`

## 五、系统特点

- **模块化设计**：符号按功能分成独立模块
- **可组合性**：符号可自由叠加，模拟多重情绪
- **双向可逆**：成对符号互为反向
- **强度可调**：`I` / `W` 让语气有强弱刻度
- **符号直觉化**：符号与含义有视觉关联
- **语法统一**：`[ ]` 包裹注音，防歧义
- **双粒度**：支持单字级和短语级标注
- **可扩展性**：开放式体系，新符号可自然融入
- **元标注能力**：`{ }` 多层嵌套
- **循环表达**：`( < > @ < > )` 简化重复性语气
- **函数复用**：`( <函数名> )` 支持定义、调用、嵌套

## 六、应用场景

- **日常聊天**：精确传达语气
- **小说/剧本创作**：为角色对话标注语气
- **配音/有声书脚本**：为配音员提供语气指导
- **语言学标注**：研究口语韵律特征
- **AI语音合成**：作为情感标签输入TTS系统
